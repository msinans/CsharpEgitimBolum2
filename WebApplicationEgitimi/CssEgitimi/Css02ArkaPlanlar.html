Css de arkaplan rengine opacity ile saydamlık, şeffaflık ekleyebiliriz.
Gerekli kod : background-color: green; opacity: 0.5;
Opaklık için bir diğer yöntem de style da renk kodu olarak RGBA kullanmaktır.
Örnek: style="background:rgba(0,128,0,0.5)"
Sondaki değer alfa dır opaklığı bu değer ayarlar

CSS arka plan tekrarı

Css ile html elemanlarına arkaplan resimleri ekleyebiliriz.

Bunun için yazmamız gereken kod şudur: background-image: url("arkaplan.gif");

CSS arkaplan tekrarı

CSS ile html elemanlarına arkaplan resmi ekleyebiliriz

background-repeat özelliği ile resim tekrarını yatayda, dikeyde veya tekrar etmeyacak şekilde ayarlayabiliriz

  1. background-repeat: repeat-y; arkaplan resmini dikeyde tekrar et
  2. background-repeat: repeat-x; arkaplan resmini yatayda tekrar et
  3. background-repeat: repeat: no-repeat; arkaplan resmini tekrar etme!

CSS arkaplan konumu

Css ile html elemanlarına arkaplan resimleri ekleyebiliriz

Bunun için yazmamız gereken kod şudur: background-image: url("arkaplan.gif");
Arkaplan resminin pozisyonunu ayarlamak için gerekli kod: background-position : right top; (center bottom, sadece center, left center, veya background-position: 18px 58px; pixellerle de ayarlayabiliriz.

CSS Arkaplan Sabitleme

Arkaplan resminin kaydırılması mı yoksa sabit mi olması gerektiğini belirtir

Eğer sayfa aşağı kaydırıldığında resmin sabit kalmasını istersek şu kodu eklemeliyiz : background-attachment: fixed;

background: #fff url('/Images/google logo.png')no-repeat fixed center bottom; kodu ile tek satırda tüm arkaplan işlemlerii halledebiliriz.
